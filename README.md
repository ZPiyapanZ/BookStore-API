# Book Store API

## How to runing the app

### On Docker
```bash
$ docker-compose up dev
```

### On CMD
```bash
$ npm install
$ npm run start
# watch mode
$ npm run start:dev
```

#### \*** Server running on http://localhost:3000 \***

## How to use API
Import the BookStoreAPI.postman_collection.json file into the Postman.<br>
You can see the api document at swagger.yaml<br>

## Note
- This project uses SQLite. <br>
- The database file is named app.sqlite. <br>
- You can view the app.sqlite file online at https://inloop.github.io/sqlite-viewer/.
